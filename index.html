Projects

I am currently working on a project utilizing Java and MySQL inspired by my latest job. I chose Java or two reasons: I have done more playing around with it than I have C++ or C#, and I don't want to admit to knowing VB; this project originally was targeted at an OS X environment while I wanted to develop from Windows and Linux. I chose MySQL as it appeared from my research to be fairly stable, widely used (and thus more likely to have someone be able to support it after I left). The company, a mom and pop operation, was founded in 1992 and had only minimally implemented newer technology since then. This means that the collections department had to manually keep track of payments, different people were generating reports from scratch based off of the same numbers and all kinds of WTF. This project was a concept I had while I was there, but never really had enough time to fully implement. Probably definitely had nothing to do with the fact that I was so busy doing redundant work. Yeeaaahhhh.
